You are given an integer N. For this mission, you should find the smallest positive number of X, such that the product of its digits is equal to N. If X does not exist, then return 0.

Hints: Remember prime numbers (numbers divisible by only one) and be careful with endless loops.

Input: An Integer.

Output: An Integer.

Example: 

checkio(20) == 45
checkio(21) == 37
checkio(17) == 0
checkio(33) == 0
checkio(5) == 5

How it is used: This task will learn you how to work with numbers.
